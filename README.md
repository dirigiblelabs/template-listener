## Overview

Simple scheduled Job with a handler writing in the console every 5 seconds.
```javascript
{
	"expression":"0/5 * * * * ?",
	"group":"dirigible-defined",
	"handler":"{{projectName}}/{{fileName}}-handler.js",
	"description":"{{fileName}} Job"}
}
```

```javascript
console.log("Hello from the {{fileName}} Job");
```
